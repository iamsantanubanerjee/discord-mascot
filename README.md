# Borphus
Your Personal Discord Server Mascot \
*Status: Under-Development*

### Key Features
* **Lightning Fast Responses:** Powered by Groq for near-instant dialogue.
* **Global Awareness:** Integrated with Tavily for real-time web searching.
* **Long-term Memory:** Utilizes MongoDB to remember past interactions.
* **Developer First:** Full monitoring integration with LangSmith.

### Requirements & Tech Stack
| Service | Purpose | Cost Tier |
| :--- | :--- | :--- |
| Discord Developer Mode | Setup Server Mascot | Free |
| Groq | LLM Inference | Free Tier Available |
| Tavily | Search Engine | Free Tier Available |
| MongoDB | Memory Management | Free Cluster (M0) |
| LangSmith | Monitoring | Free Tier Available |
| Render | Deployment | Free Tier Available |
